# Color Palette Extractor

This web app allows you to upload an image and extract its dominant color palette. The extracted colors are displayed in a visually appealing way, and you can easily copy the color codes to your clipboard.

## Features:

* Upload an image via drag & drop or by clicking to browse.
* Automatically extracts and displays the dominant colors from the image.
* View color information in both HEX and RGB formats.
* Click any color to copy its HEX code to the clipboard.
* Responsive design for mobile and desktop use.

## How to Use:

1. Click the upload area or drag and drop an image file (PNG, JPG, or GIF) up to 10MB.
2. Wait for the colors to be extracted.
3. View and copy color codes (HEX or RGB) by clicking the color swatches.
